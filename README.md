# Fake_Review_detection
# 📌 Fake Review Detection System

## 📖 Overview
This project is a machine learning-based system that classifies whether a given message/review is **fake (spam)** or **real (ham)**.

It uses Natural Language Processing (NLP) techniques to process text data and predict authenticity.

---

## 🚀 Features
- Text preprocessing (cleaning, removing noise)
- TF-IDF feature extraction
- Logistic Regression model
- Accuracy evaluation
- Real-time prediction function

---

## 🛠️ Technologies Used
- Python  
- Pandas  
- Scikit-learn  
- NLP (TF-IDF)  
- Google Colab  

---

## 📊 Model Performance
- Achieved approximately **90% accuracy**
- Evaluated using:
  - Confusion Matrix
  - Classification Report

---

## 🧪 Example Predictions
| Input | Output |
|------|--------|
| "Congratulations! You won a prize" | Fake |
| "Hey, are we meeting tomorrow?" | Real |
